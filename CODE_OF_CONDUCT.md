# Code of Conduct

Seja bem-vindo(a). Este projeto segue um código de conduta simples para garantir um ambiente acolhedor e produtivo.

Regras básicas
- Trate todas as pessoas com respeito.
- Evite linguagem abusiva, discriminação ou assédio de qualquer tipo.
- Seja profissional em críticas: foque em problemas de código, não em pessoas.

Relatar abusos
- Se você receber ou testemunhar comportamento inadequado, abra uma issue privada ou entre em contato com os mantenedores do projeto.

Consequências
- Violações podem resultar em remoção de permissões de contribuição ou bloqueio.
